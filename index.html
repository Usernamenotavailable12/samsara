How Quick Sight works

Learn more Learn more
step 01
Connect Data Sources
QuickSight allows you to connect various data sources like AWS services (e.g., S3, RDS), on-premises databases, or third-party applications (e.g., Salesforce). Once connected, you can import the data into QuickSight for analysis.
step 02
Create Datasets and Analyses
You can create datasets by preparing and transforming the imported data. Use QuickSight's built-in visualizations, such as bar charts, line graphs, and heat maps, to analyze the data. It also supports custom calculations and interactive dashboards.
step 03
Share Reports and Dashboards
After building your analyses, you can create dashboards and share them with other users or teams. Users can view the reports in real-time and interact with the data (e.g., filter, drill down) to gain insights.
